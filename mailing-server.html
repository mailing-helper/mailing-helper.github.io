使用说明 ​

欢迎使用VLink直邮营销自助下单系统，通过本文档，您将会更加熟悉设计、印刷、地址、下单、邮寄等细节。

TIP

温馨提示： 成为我们的用户请您仔细阅读使用协议； 请勿使用本平台从事违反国家法律法规的活动，如有违反其他地区或平台规则，请您知晓其中风险和后果。

模板设计 ​

设计之前，请您一定熟悉并理解对应地区规格模板需要的设计要求，请仔细核对设计稿的分辨率、像素或尺寸、相关边界线含义、特殊信息印刷区等布局要求。 设计要求在系统的"模板管理-创建模板"页面（"模板管理-创建模板-选择对应国家个规格后展示"）展示或者参考本文档。

纸张 ​

明信片均使用300GSM以上的铜版纸，正面（内容面）覆膜。

信件使用的是80GSM以上的信纸。各国信封规格：

  • 美国/加拿大：10#信封，10.48cm x 24.13cm；
  • 德法意西荷奥波：DL信封，110mm x 220mm；
  • 英国：C5信封，16.2cm x 22.9cm。
